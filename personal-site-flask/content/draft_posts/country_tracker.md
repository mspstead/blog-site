## Creating an interactive photo travel map using Leaflet, Flickr and Flask

In this post I will detail how to create an interactive travel map using geo-tagged time-stamped travel photos, this guide will follow the below structure:

1. Creating a flask website and displaying a basic Leaflet Map.
2. Populating a sqlite database with geo-tag and timestamp meta data retrieved from your travel photos using Python.
3. Creating a flask model to retrieve our photos.
4. Plotting our photos on the Leaflet Map.
5. Adding marker clustering to our Leaflet Map.
